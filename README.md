## Fiction Bot 
从biquge.com.cn下载小说到本地，并记录更新点，下次下载仅打包下载最新章节<br>
支持发送到邮箱

## 用法
```
Usage: FictionTool.py [OPTIONS]
   A tool helps you download fictions from https://www.biquge.com.cn
   to local environment or send to designated email receiver.

Options:
  --version             show program's version number and exit
  -h, --help            show this help message and exit
  -d DOWNLOADBOOK       Search for a book and return the first result in the
                        list.
  -n LASTNCHAPTER       Download the last N chapters
  -r RECEIVER           Set receiver email address
  -u FICTIONURL, --url=FICTIONURL
                        Set download URL
```
